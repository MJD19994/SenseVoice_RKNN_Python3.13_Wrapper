## Installation
To install the RKNN wrapper, follow these steps:
1. Ensure you have Python 3.13+ installed on your system.
2. Clone the repository:
   ```bash
   git clone https://github.com/MJD19994/SenseVoice_RKNN_Python3.13_Wrapper.git
   ```
3. Navigate to the project directory:
   ```bash
   cd SenseVoice_RKNN_Python3.13_Wrapper
   ```
4. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Requirements
- Python 3.13+
- Additional libraries specified in `requirements.txt`

## Getting Started
Once installed, you can begin using the RKNN wrapper. Import the required modules in your Python scripts as follows:
```python

```
